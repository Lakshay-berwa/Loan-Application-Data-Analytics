# Loan-Application-Data-Analytics
A comprehensive Statistics and Probability project using Python and a Loan Applications dataset. The project covers descriptive statistics, probability distributions, conditional probability, normal distribution analysis, skewness, kurtosis, vector operations, and data visualization for financial data insights.
# Statistical Analysis of Loan Applications

## Project Overview

This project demonstrates the practical implementation of Statistics, Probability, and Linear Algebra concepts using a Loan Applications dataset. Various statistical techniques and probability calculations are performed to analyze customer financial behavior and loan-related patterns.

The project combines theoretical concepts with Python-based implementation using popular data science libraries.

---

## Objectives

- Understand descriptive statistics
- Analyze customer income and loan patterns
- Apply probability concepts on real-world data
- Perform conditional probability calculations
- Visualize distributions using graphs
- Measure skewness and kurtosis
- Explore vector operations and linear algebra concepts
- Generate meaningful insights from financial datasets
